== Description ==



MetaboxUI can manage custom meta boxes from admin side. You can simply manage all custom field settings there. Now you can also specify custom fields for specific template files



This is compatible with OptionTree 2.4.6



A few notes about this plugin:



*   Optiontree, custom fields, metabox, metabox ui

*   Requires at least 3.0.1

*   Tested up to 4.1. Note that it might work on higher versions... this is just the highest one you've verified.

*	Added extra two option types for ease of use. Upload Media and Insert Attachment

*	added extra shortcode codes for templating [base_url] and [temp_url]

*	added shortcode support for text widgets



    



== Installation ==



1. Upload `optiontree-metabox-ui` to the `/wp-content/plugins/` directory

1. Activate the plugin through the 'Plugins' menu in WordPress

1. Manage your metaboxes from Metabox Menu located at the bottom of Navigation
